This is ARPACK with a C interface. The interface is created using Iso C Bindings
(ICB) and the source code for ARPACK is taken from Scipy [1] and is now
mainatained by opencollab's project ARPACK-NG [2]. For copyright information see
the file *COPYING* in this diectory.

External links.

    [1] https://www.gitub.com/scipy/scipy
    [2] https://www.gitub.com/opencollab/arpack-ng

To use this packahe in your C code, just run the shell-script *./build.sh* and
include the header *./INC.D/arpack.h*.
